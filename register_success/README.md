# Register Success


Ein Friendica-Addon, das neu registrierte Benutzer nach erfolgreicher Registrierung auf eine benutzerdefinierte, lokal angezeigte Erfolgsseite weiterleitet.

## Eigenschaften
* **Autark & minimal**: Keine Änderung an Systemdateien. Nutzt ausschließlich Friendicas Hook-System.
* **Flexibel**: 
  * Der Administrator kann die angezeigten Texte (Titel und Nachricht) für offene sowie genehmigungspflichtige Registrierungen direkt in den Einstellungen anpassen.
  * **Dynamische Texte**: Die Erfolgsseite erkennt automatisch die Registrierungs-Richtlinie (offen vs. administrative Freigabe) und passt die Standardtexte im informellen „Du“ darauf an.
  * **Theme-Kompatibilität**: Die Erfolgsseite erbt Schriftfarben vom aktiven Site-Theme und passt sich an Dark-Modes an.
  * **Vorschau-Modus**: Administratoren können die Erfolgsseite mit einem Klick auf den Vorschau-Button direkt testen.

---

A Friendica addon that redirects newly registered users to a custom, locally served success page after successful registration.

## Features
* **Self-contained & minimal**: No changes to core files. Uses Friendica's native hook system.
* **Flexible**:
  * The administrator can customize the success page titles and messages for both open and approval-required registration policies in the settings.
  * **Dynamic Texts**: The success page automatically detects the node registration policy (open vs. admin approval required) and displays tailored default messages.
  * **Theme Compatibility**: The success page inherits text colors from the active site theme and adapts to dark mode.
  * **Preview Mode**: Administrators can preview the success page directly via a dedicated preview button in the settings.
