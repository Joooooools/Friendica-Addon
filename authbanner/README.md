# AuthBanner

  
Das Friendica Addon Authbanner zeigt eingeloggten User bereits vorhandene Userbanner in jeweiligen Userprofil an (Banner von Friendica, Sharkey, Calckey, Hubzilla, Mastodon, Bluesky, Tumblr, etc.). Dieses Addon wurde ganz bewusst schlank gehalten, der Upload bzw. die Änderung eines eigenen Banner erfolgt unkompliziert über eine externe App wie Mona, Tusky, o.a..

Authbanner basiert auf dem Addon "Coverphoto" von Random Penguin und dem modifizierten Addon von feb.


## Installation

1. Kopieren des Ordners `authbanner` in das Verzeichnis `addon/` der Friendica-Installation.

2. Aktivierung der Erweiterung im Administrationsbereich der Instanz.
